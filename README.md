# 📡 Freelance Radar

Telegram-бот для агрегации заказов с фриланс-бирж + AI-помощник.

## 🚀 Быстрый старт (локально)

### 1. Установка

```bash
cd freelance-radar
pip install -r requirements.txt
```

### 2. Настройка

```bash
copy .env.example .env
```

Откройте `.env` и заполните:
- `BOT_TOKEN` — токен от @BotFather
- `ADMIN_IDS` — ваш Telegram ID (через запятую если несколько)
- `GIGACHAT_CREDENTIALS` — ключ от GigaChat (опционально, AI будет работать без него но выдавать заглушки)

БД по умолчанию — **SQLite** (файл `freelance_radar.db` в папке проекта). Ничего устанавливать не нужно!

### 3. Запуск — ОДНОЙ КОМАНДОЙ

```bash
python main.py
```

Это запустит **всё сразу**:
- 🤖 Telegram-бот (polling)
- 🕷 Парсер бирж (фоновый цикл каждые 2 мин)
- 📨 Нотификатор (рассылка подходящих заказов)

### 4. (Опционально) API для YooKassa вебхуков

```bash
uvicorn api.main:app --port 8080
```

## 📁 Структура

```
freelance-radar/
├── main.py              ← ЗАПУСК ВСЕГО (python main.py)
├── .env                 ← настройки
├── requirements.txt
├── core/
│   ├── config.py        ← конфигурация
│   ├── database.py      ← SQLAlchemy (SQLite/PostgreSQL)
│   ├── models.py        ← ORM-модели (12 таблиц)
│   ├── gigachat_client.py ← GigaChat AI
│   └── yookassa_client.py ← платежи
├── bot/
│   ├── keyboards.py     ← все клавиатуры
│   ├── middlewares.py   ← регистрация, бан, streak, лимиты
│   ├── states.py        ← FSM-состояния
│   └── handlers/
│       ├── start.py     ← /start, /help, кнопки меню
│       ├── filters.py   ← ⚙️ фильтры
│       ├── ai_handlers.py ← 🤖 AI, сохранить, скрыть, CRM
│       ├── profile.py   ← 👤 профиль, тихий режим, ачивки
│       └── admin.py     ← 🔐 админка (полная)
├── parser/
│   ├── base.py          ← BaseParser + OrderDTO
│   ├── engine.py        ← сохранение в БД, запуск
│   ├── matcher.py       ← матчинг заказов ↔ фильтры
│   └── parsers/         ← habr, fl, freelanceru, kwork, workzilla
└── api/                 ← FastAPI (опционально)
    └── routers/         ← orders, stats, payments webhook
```
